public enum FileOverwriteMode

Whether to overwrite a target file. Used with FileInfoExtensions.RequiresOverwrite(System.IO.FileInfo, System.DateTime, FileOverwriteMode)

Members

Member Description
Always Always overwrite the file.
IfNewer Overwrite if the file doesn't exist or if the file's last write time is older than a given time.
Never Never overwrite an existing file.
